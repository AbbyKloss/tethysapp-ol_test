How to Use:
Lake Area Sliders

The sliders at the bottom of the screen all set a threshold for the lake area of each point. You can change the threshold by slowly moving any of the sliders. The rightmost one is recommended for this, as it has been somewhat tuned for the dataset presented here. Any points that are red have lake areas that are above the threshold, any points that are green have lake areas below the threshold.

Point Graphs

Clicking on a point creates a popup with the point ID, the lake area of that point, and a hydrograph for that point. Clicking the button will also show the graph, as well as an option to disable the popups or view details. You may also select multiple points by clicking and dragging the mouse while holding Ctrl (âŒ˜ on Mac), although no graph will be shown for these points.

Changing Point Coordinates

You are able to move each point around by selecting a point then clicking and dragging the point around. Changes are saved automatically. The button shows changes made since the start of the session.

Save Coordinate Changes

Changes are saved automatically as points are moved. You can click Update Changes below to refresh the page and see any changes that were made elsewhere.
